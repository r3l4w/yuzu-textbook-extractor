## 📖 What is yuzu-textbook-extractor?

yuzu-textbook-extractor is a Chrome extension designed to help you get textbook sections out of Yuzu Reader. It lets you save these sections as easy-to-print PDFs or as Markdown files. The Markdown files include LaTeX math formulas and images that show up just like in your textbook. This makes reading, sharing, or studying your textbooks more flexible.

This tool works directly in Chrome. There’s no need to run complicated software or learn how to code. If you can install a Chrome extension, you can use this to keep your study materials organized and accessible.

---

## 🛠️ Key Features

- Extract specific parts of textbooks from Yuzu Reader.
- Save sections as ready-to-print PDFs.
- Export content as Markdown with LaTeX math formatting.
- Keep embedded images intact in your exports.
- Works smoothly within the Chrome browser environment.
- Supports ebooks in epub format through Yuzu Reader.

## 📄 How to Use yuzu-textbook-extractor

Now that you have the extension installed, here is how to extract textbook sections.

### Step 1: Open Yuzu Reader

Go to your Yuzu Reader page and open the textbook or ebook section you want to save.

### Step 2: Activate the Extension

Click on the yuzu-textbook-extractor icon in your Chrome toolbar.

### Step 3: Select Textbook Sections

Use the extension’s tools to highlight and select the text you want to save. You can choose entire pages or smaller parts.

### Step 4: Choose Export Format

Pick how you want the content saved:

- **PDF** for easy printing or offline reading.
- **Markdown** for editable text, complete with LaTeX math formulas and embedded images.

### Step 5: Download or Save Your File

The extension will process the content and create your file. You can then save it to your computer or open it directly.
